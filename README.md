# sturm_diskurs
A visualization of Leonhard Christoph Sturms intellectual network. It was created using Gephi (https://gephi.org/) and the SigmaExporter-Plugin (https://github.com/oxfordinternetinstitute/gephi-plugins/tree/sigmaexporter-plugin). The underlying data was harvested from the following databases: VD17 (http://www.vd17.de/), VD18 (https://kxp.k10plus.de/DB=1.65/), Kalliope (http://kalliope-verbund.info/gnd/117364177), Gelehrte Journale und Zeitungen als Netzwerke des Wissens im Zeitalter der Aufkl√§rung (https://adw-goe.de/gjz18/datenbank/), and Personen- und Korrespondenzdatenbank zur Leibniz-Edition (https://leibniz.uni-goettingen.de/persons/view/987).
